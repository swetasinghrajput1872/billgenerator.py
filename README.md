# Bill Generator using Python & MySQL

This project is a simple **Bill Generator** built using **Python** and **MySQL**. It takes order details from the user, stores them in a MySQL database, and generates a text file bill with order information and total amount.

## Features
- Takes order details from user input  
- Stores data in MySQL database  
- Calculates total bill amount automatically  
- Generates a `.txt` bill file with date and time
